# 项目名称
音沐APP

## 项目描述
 这是一款手机端的音乐播放软件

## 项目如何执行
- git clone https://github.com/GasolineRemon/YinMu.git
- cd YinMu
- npm install
- npm run dev 


## 项目中所使用到的技术
- vuex
- vue-swiper 
- vue-cli 
- vue-router 
- express
- axios
- mint-ui
- vue-awesome-swiper
- less、less-loader、style-loader
- element-ui

## 项目所遇到的问题及解决方案
### 在同级组件之间传递参数的问题，使用axios解决，通过在路由上传递参数，再通过params接收
### 多级路由在进入后不能直接返回上一级，使用路由传递参数，返回时携带参数改变原先的默认值
